# web-quickstart

Multi-agent infrastrucure quickstart for 2025 web apps.

## Supported IDEs

- Project IDX
- Cursor
- VS Code

## Stack

genkit 1.0, MCP, vertex-ai + open-ai (lite-llm), zod, hono,
astro 5 + server islands + mdx, drizzle,
react, tailwind, shadcn, framer motion,
nanostores

## Local Development

```bash
pnpm install
pnpm dev
```
